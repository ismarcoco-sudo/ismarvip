# Ismar VIP - Despedidas de Soltero en Cuba

Sitio web premium para la organización de despedidas de soltero en La Habana, Cuba.

## Estructura del Proyecto
- `index.html`: Página principal con diseño premium y optimización SEO.
- `/assets`: Imágenes generadas y recursos visuales.

## Despliegue
Este es un sitio web estático. Puede ser desplegado en cualquier servidor web (GitHub Pages, Vercel, Netlify, Hostinger, etc.).

Si el servidor requiere un comando de inicio, puede usar:
`npm start`
